# SQL_Pizzeria_Project
This repository contains a SQL-based analysis of a pizzeria dataset, exploring sales trends, customer behavior, inventory management, and staff performance. By leveraging SQL queries, key insights were extracted to optimize business operations. The project encompasses data cleaning, exploration, and in-depth analysis to uncover actionable recommendations. Key findings include top-selling items, customer segmentation, and inventory optimization strategies. The code and results are presented for reproducibility and further exploration. This project demonstrates proficiency in SQL, data analysis, problem-solving, and provides a valuable foundation for understanding business operations through data-driven insights.
